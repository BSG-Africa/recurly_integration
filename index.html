Recurly Integration

A Java Client Library for Recurly

View project onGitHub

Recurly.js is a Javascript library which allows you to easily embed and customize PCI compliant forms within your website. HMAC digital signatures are used to verify information sent to Recurly via your customer's browser. The signature validates that parameters you have specified in your HTML have not been tampered with before being received by Recurly. The client library consists of helper methods to create and verify these signatures.

Using the library

With the Recurly Integration Library being in the Maven Central Repository, its simply to use it in a maven project. Simply add the necessary dependency to your project POM. See the project's README on GitHub for an example. Not using Maven? The library JAR is available on the project's download page.
